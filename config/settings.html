Styling Panel

Turn on/off Styling Panel

Default is off (unchecked). Turn on (check) to play around with layout/font-size/font-family/colors etc on the webpage. However, visitors to the site will also see these buttons. So maybe do it during maintenance hours and 'hide' the website by setting a password and message in /Settings/General
Typography General Font families uploaded here will be accessible in all pages. Font sizes are in percentages, 1 is the same as 100% which defaults to 16px, 2.5 is 250% of 16px, so 40px. Also note, most of the values entered below are optional, there are default sizes, colors, and styles for all fonts, so you do not HAVE to enter a value.

Google Fonts

Please copy and paste the Standard link (NOT @import nor Javascript), the link should look like this: <link href='http://fonts.googleapis.com/css?family=...' rel='stylesheet' type='text/css'>
